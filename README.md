# 한솔다골프 회원권 관리 대시보드

골프 회원권 예약 및 임원 배정 현황을 관리하는 대시보드입니다.

## 파일 구조

```
├── index.html   # 메인 페이지
├── style.css    # 스타일시트
├── app.js       # 데이터 및 기능 로직
└── README.md
```

## 실행 방법

`index.html`을 브라우저에서 열거나, GitHub Pages로 배포하면 바로 사용 가능합니다.

## GitHub Pages 배포

1. 이 저장소를 GitHub에 push
2. Settings → Pages → Branch: `main`, folder: `/ (root)` 선택
3. 저장 후 제공되는 URL로 접속

## 기술 스택

- Chart.js v4.4.1 (CDN)
- chartjs-adapter-date-fns v3.0.0 (CDN)
- 순수 HTML / CSS / JavaScript (프레임워크 없음)
